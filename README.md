# 服務處 CRM 第二版

這是可在手機與電腦瀏覽器操作的 PWA 原型。

## 已完成
- 儀表板
- 民眾資料
- 一人多案件
- 案件狀態、類別、承辦人、處理紀錄
- 追蹤任務
- 今日與逾期提醒
- 里別、狀態、類別、承辦人統計
- JSON 完整備份與還原
- 民眾與案件 CSV 匯出
- 離線快取
- iPhone 加到主畫面

## 使用方式
本專案需要透過網站伺服器開啟，不能只在 Dropbox 裡直接點 index.html。

最簡單的免費部署方式：
1. Netlify Drop
2. Cloudflare Pages
3. GitHub Pages

部署後，用 iPhone Safari 開啟網址，點「分享」→「加入主畫面」。

## 目前限制
- 本版使用瀏覽器本機儲存，尚未接雲端資料庫。
- 尚未加入 Google 登入、多人權限、附件上傳與 Google Calendar。
- 正式存放真實個資前，必須做登入、權限、加密、備份與操作紀錄。

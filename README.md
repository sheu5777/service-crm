# 服務處 CRM V7 功能擴充版

## 本版新增
- 民眾：備用電話、鄰別、生日、LINE、Email、標籤
- 一鍵撥電話、Google Maps 導航、Email
- 案件自動編號
- 建立日期、優先級、結案日期
- 統計報表：狀態、類別、里別、承辦人
- 民眾 CSV 與案件 CSV 匯出
- 保留 Google 登入、Firebase 雲端同步、角色權限與操作紀錄

## 更新
把本資料夾全部檔案上傳到 GitHub `service-crm` 並 Commit。
V7 沒有更改 Firestore Security Rules，因此不需要重新貼規則。

## 個資提醒
本系統定位為民眾服務與陳情案件管理。不要新增政治傾向、投票意向、支持度、健康資訊或其他非必要敏感分類。
